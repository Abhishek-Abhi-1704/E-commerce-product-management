🛒 E-Commerce Product Management System

This is a simple **E-Commerce Product Management Application** built using 
**React (Frontend)** and 
**Spring Boot (Backend)** with **H2 Database**.  
The application allows users to manage products by performing basic CRUD operations with product images.


### Frontend
- React.js
- Axios
- HTML / CSS / JavaScript

### Backend
- Spring Boot
- Spring Web
- Spring Data JPA
- Hibernate
- Lombok

### Database
- H2 In-Memory Database



## Features

- Add a new product with image
- Update existing product
- Delete product
- View all products
- Upload and store product images
- REST API integration between React and Spring Boot
